About the Project

Built to protect people
from online harm

Peacemaker is a final-year B.Tech project that uses state-of-the-art deep learning models to identify cyberbullying and hate speech across multiple input modalities. From a typed comment to an uploaded meme to a voice recording — Peacemaker analyses it all and returns a verdict within seconds.

Core Capabilities

Five ways to detect harm

Each modality runs through a dedicated model pipeline, with results fused into a single toxicity verdict.

💬

Text Analysis

Fine-tuned DistilBERT model classifies raw text, comments, and captions for cyberbullying patterns with high confidence.

DistilBERT
🖼️

Image Analysis

CLIP's zero-shot vision-language model identifies violent, threatening, or hateful imagery — even when overlaid with text.

CLIP · ViT
🎙️

Audio Analysis

OpenAI Whisper transcribes spoken audio to text, which is then passed through the DistilBERT classifier for toxic speech detection.

Whisper
📸

Instagram Scanner

Enter any public Instagram username and Peacemaker fetches and analyses their recent posts and captions for toxic content patterns.

Instaloader
✈️

Telegram Scanner

Analyse public Telegram channels for hate speech and bullying across both text messages and shared images.

Telethon
⚡

Real-time Results

Flask REST API backend delivers predictions in real time with confidence scores and threat level indicators.

Flask · REST API

Text Analyzer

Think someone sent you a hurtful message? Paste it here and Peacemaker will tell you if it counts as cyberbullying. It works on comments, captions, DMs — anything written.

Image Analyzer

Received a scary meme, a threatening photo, or an image with hateful text on it? Upload it here. Peacemaker uses a powerful vision AI (CLIP) to understand what the image actually means — not just what it looks like.
